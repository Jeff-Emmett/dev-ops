Multi-Location Support

This setup is designed to be deployable at any physical location. Each node joins the Headscale mesh network and automatically becomes a failover target.

LocationRoleRequirements
Home (primary local)AI inference + NAS + media + warm standbyStable internet, 50+ Mbps upload, 10GbE LAN
Friend/familyOffsite backup nodeAny internet, always-on power
Coworking spaceAdditional failoverEthernet port, power outlet

To deploy at a new location:

  1. Assemble same hardware (or a subset — even a Raspberry Pi works as backup-only node)
  2. Run ./setup/01-base-install.sh — installs OS, Docker, joins Tailscale mesh
  3. Run ./setup/03-backup-target.sh — configures as Restic + PostgreSQL replication target
  4. Optionally run ./setup/05-warm-standby.sh for full failover capability
